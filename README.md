Guideline:

- when you edit a script...
- TELL EVERYONE (so we don't accidentally overwrite it)
- Make sure you add comments while you are making changes (easier that way)

- Don't change header files really
- (unless you want to add a new global var to main.h)

- compile like this:
- gcc .\main.c .\main.h .\battle.c .\battle.h .\ui.c .\ui.h .\events.c .\events.h

- for the github: (kind of a long process to get to this point tho)
- git commit -am "initial commit"
- git push origin master
